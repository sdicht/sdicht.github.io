# Paper 1: Modernism in Context

## Task
Write a 1,200–1,500 word essay analyzing one text from Weeks 1–3, focusing on how it embodies (or resists) modernist aesthetics.

## Requirements
- Include at least two critical sources.
- Cite all references using MLA style.
- Submit via email as a PDF by **Oct 12, 2025**.
